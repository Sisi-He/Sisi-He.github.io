- No current TOEFL/IELTS score

- Make the application research-driven, not development-driven
- Use the personal website as a research portfolio, not only a CV page
- For each project, explicitly state:
  - research question
  - system contribution
  - AI/XR pipeline
  - evaluation or user study
  - findings or impact
  - personal contribution
- Secure strong letters from Tsinghua and HKUST(GZ)
- Prepare TOEFL/IELTS early
- Build a faculty-fit matrix before writing SOPs
